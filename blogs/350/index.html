Markdown常用命令笔记

Dec 30th, 2011 | Comments

今天又复习了下markdown，既然学了，就该好好学一下，还做了点笔记写在纸上，只是随便写的，怕到时候丢了找不到，就记到博客上吧。

  1. 链接

    <url>


    [文字](url)


    [文字](url “文字”)


    [文字][标记]


    [标记]: url　“文字说明”

  2. 标题

    ##### 标题
    标题
    ==

  3. 粗体斜体

    文字 文字
    文字 文字

  4. 粗斜体

    ***文字*** ___文字___

  5. 引用

    >文字

  6. 原文输出

    `文字`

  7. 列表

    * 列表１

  • 列表２
    + 列表３
    - 列表４

    1. 转义

      \

    2. 换行

      句子后两个空格

    3. 段落

      空行

    4. 水平线

      ——————-
      *



      这篇笔记就是在vim-vimpress下使用markdown语言编写，但是小标题的文字并没有被加大加粗。


      找到原因了，和vimwiki的语法还是有点不一样，小标题应该用#才行的。

Ipad2一年

May 25th, 2012 | Comments

去年五月份国行买了ipad，一转眼一年了，时间过得真快，在这一年里ipad是除我手机和电脑之外使用最多的电子产品了，可怜的kindle在边上落灰，为了挽回损失，我也要多看几本书啊。

去年焦急的等待ipad的到来，了解了苹果的销售，认识了嘉里大通物流，明白了苹果的售后，第一时间拿到ipad时，兴奋的睡不着觉，折腾ios5系统，等待4.3.3的越狱，可是一年后我对这些一点也感冒了，系统有ota有更新到最新的吧，不管它能不能越狱，反正和我工作相关的正版软件已经都买好了，也只的苹果引导了我开始购买正版软件（太贵的商业软件除外）。

ipad用得挺多的，刚开始无论到哪都会带着，爬山，坐车，约会等等。刚开始，什么facetime，icloud，find my phone，phote booth离线地图等等新鲜玩间都试了个遍。慢慢的它在我手上的功能开始退化，现在的主要功能就是不想开电脑时在上面用自带的浏览器浏览下网页，用instapaper看下想要细读的文章，用mobile rss订阅google reader回家细细的品，还有最最主要的功能就是在avplayer里拷上几部电影躺床上看。有时实在是不想带电脑就用goodreader看下文献，用iworks翻翻文章，什么upad等手写软件基本上是没用过，亏我还花了100多块买了支电容笔 ，要是现在根本不会买。

列下我买过的正版软件吧，大部分都是比较实用的

游戏
+ Fruit Ninja HD
+ Fruit Ninja: Puss in Boots HD
+ MOMOPOLY for iPad
+ Need for Speed Hot Pursuit
+ Grand Theft Auto 3
+ Asphalt 6: Adrenaline HD

工作
+ DocAS-Annotate PDF, Take Notes and Reader
+ Instapaper
+ Splashtop Remote Desktop for iPad
+ Echofon Pro for Twitter
+ IM+ Pro
+ UPAD
+ Goodreader for iPad
+ AVPlayerHD
+ Miu Term HD
+ iMovie
+ iphote

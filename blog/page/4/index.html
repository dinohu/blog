最近读的4本书

Nov 26th, 2011 | Comments

列过一个书单说最近买了不少书，买来总要看吧，否则又是堆在那里落灰浪费了。我以前不是怎么喜欢看书的，但受到@MapleShadow 枫影的影响，他每周会发一些读后感，还有和TAN老师住的时候他买了不少书，加上现在回到住的地方没有网络，早晨起来前，午睡前，晚上睡觉前翻几上几页，刚好可以促进入睡。因为没有大量时间来完成整本书的，所以看的书都是断断续续的。

《你若安好，便是晴天》

这本书是写林徵因的，我最初了解到她是在电视上，她住过的房子没有被列为重点文物保存下来，看的时候就顺便查了下这位才女，刚好母亲在我后面看到了问她是什么人，我也就详细看了下，这次买书就买了本他的传，《你若安好，便是晴天》。书买来的时候是办公室Ding老师帮我拆的，她很喜欢这本书的包装，淡黄色，特别是书名的几个字，你若安好，便是晴天，更是有感觉。那么多本书中之所以先看这本书就是想看看这位才女到底如何有才，书的前半部分写得很有感觉，特别是林和徐的康桥之恋，后半部分就比较平淡，再加上我看不太懂现代诗，大部分都跳过了。书中印象最深的一句就是梁问林：“有一句话，和只问一次，以后不会再问，为什么是我”林答：“答案很长，我得用一生去回答你，准备好听我了吗？”多么巧妙的回答啊！也就这样的人才能做做到。看完这本书想改签名档时，发现QQ中一位好友已经将其签名档改为“你若安好，便是晴天！”莫非也在看这本书？

《历史不忍细看》

一位散文家写关于历史的，对作者不感兴趣，这本书主要看内容，书中基本上讲的一些名人轶事，都是以前历史书上了解不到的，但是整本书看下来基本上差不多围绕着战争，然后一些不为人知的事，按朝代上宋明清写较多，也许书本纸张有限，只能写这么多吧，看到最后就是觉得有点乏味了，然后就挑着看，慢慢把书看完。

《去，你的旅行》

半年前看过一本《背包十年》，看这本书是以为会和那本差不多，但是看完发现这是一本旅游手册再加上自己一点情感的书，但是书的最后一段最感人，就是写父母的，越是长大，越是理解父母。书中作者总是写到一种咖啡拿铁，意大利浓缩咖啡和牛奶的经典组合，我这种从来不喝这种抗困饮料的人是不知道的，刚还随手去查了下。当然我也不喝荼，除非到饭店或者到别人家不得不喝时才喝。平时都是白开水，向老蒋学习吧，现在防止增肥也很少喝碳酸饮料了。书中最喜欢的一段，也就是我觉得旅行的意义：“我一直记得长途火车的味道，一站又一站无法入眠，偶尔停在某个异乡的小镇便下车去抽支烟，然后在火车关门前丢掉手中半支烟在夜空里。下一站？自已都不知道，那种快乐肯定是现在的豪华飞行或者高铁无法比拟的”。有时候我就觉得旅行就应该用最简单最省钱的方式去享受旅途中的乐趣。

书的排版不太喜欢，可能用惯TeX的源故 ，觉得这个排版实在太乱了，经常用照片充满整页，字太小，行间距估计有2倍，难道作者没东西可写了么？以这种方式增加厚度来赚读者的钱是不对的。

《当我谈跑步时我谈些什么》

这本书是在＠terrixchan 特暖暖的发推时看到的，是一个中国人翻译，村上村树写的。高考完看过村上的《挪威的森林》，只知道他是个比较优秀的作家，没想到他还喜欢跑步，除了下雨天外每天都要跑，而且每年至少跑一次马拉松，这人的毅力实在太强了。读完这本书也许作者的意思就是做事一定要坚持，日复一日，就像吃饭一样，总会有成果的。对于跑步，如作者所说，跑步不适合所有人，就如我，体重大了膝盖实在是受不了。这本书是看的电子版，如果有机会可能要买本纸制的收藏。

下一本书，早上的时候已经选好了，季羡林老师写的《影响我最大的几本书》，我读书比较慢，希望花半个月能看完了。

关于工作和选择的两篇好文

Nov 19th, 2011 | Comments

 

 

最近一直很忙，有两篇文章看了很久了一直想写点读后感，可是由于实验太忙，没有时间来写。

  1. 如何做一名优秀的博士生：（一）时间的付出
    http://blog.sciencenet.cn/home.php?mod=space&uid=46212&do=blog&id=484416
  2. 如何做一名优秀的博士生：（二）方法论的转变
    http://blog.sciencenet.cn/home.php?mod=space&uid=46212&do=blog&quickforward=1&id=486270

 

上面的两篇文章《如何做一名优秀的博士生》我是在师兄的博客上看到他转载的，在他那里合成一篇，如果引用他的觉得有点不合适，就在网上找了下原来是两篇。这两篇文章看了好几遍，然后自已前几年博士生涯去对比，发现自己的博士就读得稀里湖涂一点没有方向。

 

对于文章中所提到时间的付出，作者真是在科研上花了不少的时间才可以这么说。想想自己前几年在学习上的时间花得实在太少了，虽然也学到不少知识，但是对现在工作应付起来还是有点吃力。对社会上很多职业来说，想要成为行业的佼佼者，都必须付出大量的时间。觉得现在做科研，只在脑袋不是太笨，再稍微有点想法，然后努力工作个十几年二十几年，差不多也可以拥有本领域较高的水平。就像我导师，现在把大部分的时间都花在了科研上，每天晚上干到一两点，早上六点多起床，中午睡两个小时，除去睡觉吃饭等时间估计有14个小时以上，但是他还觉得时间不够用。以前曾经在电脑上装过一个可以记录各软件使用时间的软件，经过一个月的统计，才发现自己每天花在学习上的时间不会超过3个小时，也难怪自己现在很多东西都不懂，时间花少了。TAN老师也和我说过，对于我以后的道路想要做出水平，光学5年，机械5年，电子学5年，合在一起至少也要学个10年才能达到一个一般研究员的水平，反正就是熬吧，慢慢熬，媳夫也会熬成婆。

 

对于方法论的的转变，这点对这几年的我影响太大了。从小学习东西都是系统式的学习，拿着课本，跟着老师从头到尾学一遍，然后考试，凡是要考的东西书本上都可以找到。可是现在的工作，很多问题都是没有答案的，或是多学科的交叉，跟本不可能一样样的系统学习，而是现在比较流行的查找式学习，用到什么去找什么，但是这都建立在有比较扎实的基础上，这也就需要时间上的付出。现在对于快速学习能力要求非常高，这点就比老板差太多了，看来自己还要继续磨练。文章中说的批判性分析，现在大部分东西早在几十年前就已经有人做过，要想有所突破，就需要批判性分析，凡是都要问个为什么，自己怎么做也要讲出个道理。由于这点做得不够好，也老是被老板说为什么要这么做，别人为什么要那么做，别人那么做肯定有他的道理。而我以前就随心情的好坏选个条件做就好了，也难怪做得不太好。

 

科研文献和学术讲座的取与舍，科大和所里经常有这种牛人来做学术报告，以前也经常每个都去听，但是听得多了，发现忽悠的人占了大多数，真正踏踏实实做科研的人实在是太少了。难怪老板说有人邀请他会议被他拒了，他说和那些人交流还不如在家多看几篇好文献。说起这些会议，如果是两天的话，报告半天，游玩1天，半天车上，所以说大部分会议都是打着开会的幌子去旅游。

 

最后就是跟着逻辑走，老板也经常和我们说逻辑，经常能抓到我们的错误，也经常看他在学习答辩的时候把学生折磨的半死，他只要在那短短的报告时候内抓出论文中站不住脚的地方，一个是多靠年的经验，对于自己不懂的领域就去抓逻辑错误，真是一抓一个准啊。所以，Follow logic!

  1. 路是自己选的，别问。
    http://www.lixiaolai.com/archives/11136.html

这篇文章是在笑来老师博客中看到了，下载下来也是看了好多遍。如果早些日子看到这篇文章，在很多地方取舍时候就不会变得很痛苦。就像笑来老师说的，要自己做决定，只有自己才能对处境更加清楚，即使和别人把情况说得再清楚，也会有点信息不对称，去问别人吧，别人只能给个建议或者意见，但是还要自己选，因为承担后果的永远是自己。实在做不出选择，那就抛硬币，也要自己抛。

Linux下安装gaussian09

Oct 28th, 2011 | Comments

最近想用gaussian算点东西，windows下的效率实在太低，于是想在自己电脑上试试，笔记本上是新装的ubuntu11.10，去师兄那拷了gaussian09的linux版本来安装，从网上查了下说安装很简单，但是我还是折腾了好几天，主要是一个不起眼的地方出错了，也不知道是不是单机版的原因。

从网上查了不少安装办法，大致都是这个样子，顺便引用下师兄的文章：

http://blog.csdn.net/tengh/article/details/5996150

高斯在linux下的安装很简单。首先把gaussian09的文件解压到目标文件夹。比如

/home/zhuh/

然后设置环境变量，如果是root用户，修改 /etc/profile文件，如果是一般用户，修改/home/.bashrc文件。

添加以下几行：

 

export GAUSS_EXEDIR=/home/zhuh/g09/

export G09ROOT=/home/zhuh/g09/ export

GAUSS_SCRDIR=/home/zhuh/tmp/g09/

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/$G09ROOT


其中/home/zhuh/改成自己的路径。然后保存。

在命令行更新环境变量，root用户输入：

source  /etc/profile

普通用户输入：

source  /home/zhuh/.bashrc

/home/zhuh改成你自己的用户路径。

我按照这个设置，用了很多次运行就报错，报错内容如下：

Entering Gaussian System, Link 0=g09/g09

PGFIO/stdio: No such file or directory

PGFIO-F-/OPEN/unit=11/error code returned by host stdio – 2.

File name = /tmp/g09/Gau-3850.inp

In source file ml0.f, at line number 182


从上面的内容看也看不出什么错，然后师兄提醒我这样安装
1、./configure

2、make

3、make install


还是不行，文件夹里根本没有./confiure文件，快下班时候，师兄又过来给我折腾了半天还是不行，我都快崩溃了，已经放弃在linux使用gaussian，但最后还是想试下gaussian03行不行，于是又在网上随手搜了几篇gaussian03的文章
http://hi.baidu.com/curioz/blog/item/6324b37eb94b6e300dd7da6f.html

http://www.baisi.net/thread-14466-1-1.html


发现这两篇文章下面临时文件夹只是/tmp，于是就把g09去掉，然后按照前面的方法又来了一篇居然行了。

现在贴出我的配置

export GAUSS_EXEDIR=/home/huzhenhua/g09

export G09ROOT=/home/huzhenhua/g09

export GAUSS_SCRDIR=/tmp

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/$G09ROOT

export PATH=$PATH:/home/huzhenhua/g09


 

再装Ubuntu11.10

Oct 23rd, 2011 | Comments

已经下决心把硬盘全格掉装ubuntu了，以后在自己的电脑上就一个系统Ubuntu，我想足以应付我手头上大部分事情。

 

准备工作：

主要是备份文件和制作U盘启动盘，备份资料前两天已经完成，下面就是制作启动盘，由于手头没有linux操作系统，只好在win下用ultraISO烧录了，这个过程还算比较顺利。以前安装的时候都是用alternate版本的，网上虽然查了这个不能用U盘安装，我还是想试试，结果刚分完区就出问题了报错不能安装，也不知道怎么回事。又下desktop的版本安装，还好用USTC的源下镜像几分钟就搞定，再用ultraISO烧到U盘，进入U盘系统，一切都很熟悉，画面变得漂亮多了，我就知道现在叫gnome3什么的，主要是太久不观注了。

 

开始安装：

U盘desktop安装很方便，反正我是就一个硬盘装一个系统，不用考虑什么grub的问题，硬盘分了两个区，一个/分区30G，一个home分区120G，都是ext4的，现在有3G内在swap也就不要了。这次装系统不想像以前一样挂7，8个分区，因为以前觉得全部分开到时候重装系统时候会很方便，什么软件都不要重装了，但是一次也没用上，该重装的还是要重装，缺什么库文件太多了，也懒得折腾。

 

安装过程很多，几分钟就搞定了，好久不用有点手生了，记得上次用ubuntu还是9.04的时候，一下子两年过去了。装完系统第一步就是加源，虽然我记得要做这件事，但是那source.list放在哪都找不到，只好再网上查，顺便又加了ustc的源，更新了下系统，然后装上极点五笔输入法，这样基本上我就能用了，实在不习惯拼音输入了。

 

现在很懒，懒得折腾，搜到了一个ubuntu总结得比较好的个人网站，然后就跟着他的设置再按照自己需要安装。主要是参考了下面这个笨兔兔，还有一些结合了下forum.ubuntu.org.cn论坛的。

 

笨兔兔

ubutnu中文论坛

很大一部分软件都是能过ubuntu软件中心装的，只要能想到基本都有，比以前时候方便了很多，系统自带的是libreoffice，用着不太习惯，刚好院里是买了有永中的office，可是由于我安装的64位系统， 怎么搞也装不上，说少个什么64位的库文件。下面给出永中office的链接。

FQ：

基本上还是老套路，先装Gstd用ssh设置代理，然后再用firefox+autoproxy先设置成功，装chrome＋switch proxy，设置很方便，但是又遇到以前的问题，chrome经过一次上网后生成pac文件就不行了，还好在商店里又找到switch proxy plus，这个能用，很高兴，结合通道还很方便。VPN经过多次折腾也登录不了，不知道是pptp的问题还是怎么回事，能FQ了也就不管了。

 

 

常用的软件不太多，主要就是BBS，从在线商店里装了Gemanx，设置字体很丑，于是又想折腾字体，记得以前弄字体经常把系统搞挂，没想到现在这么方便，把windows下字体拷过来双击就安装了，真是一大改进啊！

 

本来对在ubuntu下用苹果的设备不报有什么希望的，也就弄了个虚拟机vbox，这个vbox还是以前弄的才10G大小，根本扛不住的备份了。虚拟机的增强包没装上，所以外接USB在虚拟机里也没识别，只好做罢，不过还能淘宝一下也挺好的。把ipad插电脑上，居然挂载了两个分区，一个是可以看到系统文件，我没敢对里面东西操作，另一个就是软件，可以看到ipad上装每个软件的图标，点击图标可以看到里面的文件，尝试拷了几个文件进去，居然能用，真是太赞啊，这下我不用itunes也基本能用了，就是拷入速度慢了点，才6.5M/s左右。

 

还折腾了很多东西，安装过了好几天了也记不太清楚，想到点加一点吧。

 

无力吐槽

Oct 21st, 2011 | Comments

最近很累，身心疲惫，好久都没有这种感觉了。虽然每天都能保证8小时的睡眠，但是还感觉睡不够。每天都感觉好多事要做，但总是有做不完的事。我自认为算是有计划的人，但总是跟不上别人的节奏，也许说的人根本不知道做得人的辛苦。

从过了十一以后都很忙，先是XX基金的答辩，本不抱有任何希望，但评审算是公平，过了第一轮，后面的事也就不归我管。基金中了最好，可以给自己贴贴金，不中也罢，那钱和我半点关系也没有，我只要干好自己的事就好。

刚好基金有点眉目，实验开始启动，状况比我想象的要差很多，就是要把别人丢下的烂摊子玩转起来，反正我知道这东西自从我到这里就没有人用过。虽然说课题组有些研究基础，但还是需要花些时间的，还好有人带这玩，自己玩就歇菜了。

开始收拾这幅烂摊子，没想到会如此的麻烦，机械泵分子泵换油，整理和蜘蛛忘似的电线，更换屋子的排气管道，收拾仓库打扫卫生等等，没一样活不需要我，拧螺丝拧得手也起茧，悲惨的生活。

这辈子都没想过装win98系统，可是科大给我们做的数据采集卡，只能用在98/95系统上。系统崩溃掉了，怎么也进不去，把硬盘拆下来挂在其他电脑上发现系统windows文件夹都没了，这应该不是病毒干的，不知道哪个煞笔拷数据时候给剪切走了。系统没那么好装，最早的win98用软盘引导的，还好找到了能引导的iso刻盘安装，经过多次尝试，发现TMD硬盘有问题，win自检都通不过。格式话硬盘又花了一个下午安装好系统发现原来的软件都不干活了，本不报有什么希望的时候在备份盘里发现了05年做的ghost，我想应该是能用的，因为这套系统在05年就已经搭建好了。找了张PE光盘，无奈电脑配置太烂，等了N久才进到PE里，索性恢复都很正常，进入系统后软件基本干活了。看来这系统要一直用下去，什么库文件太难弄了。

再说硬盘备份数据，两个80G的硬盘共分了8个区，每个盘20G，搞得每个盘里都是数据和软件，当时怎么不少分几个区，难道不知道硬盘只要文件和文件夹的概念。软件是在这破电脑上写的，很多还是用的绝对路径，想把它们挪个地方也别想，我也不想看他们写的恶心的VC程序。硬盘都快满，每个盘都剩点，数据全删掉又可惜，备份又太慢，选择性删除了些。再说软件备份，我发现硬盘里有无数个备份，都不知道那个可以工作，而且文件夹名全是新建文件夹。以前和他们说过版本管理软件，可他们觉得使用日期加编号再打包备份更好，这下可苦了我。

数据采集卡测试，以前我折腾软件较多，对硬件一知半解，只能硬着头皮上。开始用时序控制器触发啥都没看到，接着试了其他三张破卡毛都没有。第二天拿信号发生器和示波器一起测试，看啥也看不到，到我都快绝望的准备换新卡的时候软件出现信号了，下面可有事做了。我不知道他们以前是怎么测试的，连个测试报告也不留下来，按照自己的想法把能做的做了一遍，还是被训了一通，我都出离的愤怒了，要不是我脾气好，早甩东西走人了。这样的是前几天也发生过一次，我把电线接得好好的，为了区别每根线还特意贴了标签，后来还是被发了一通牢骚，当时气得不得了，还是活生生把气咽下去，真感觉到为人打工不是那么好干的。

给ipad升级ios5，GM版出来很久都没升级，这次正式版发布实在忍不住了，几个新功能很诱人，反正我也买了些常用的软件，基本上不影响我正常使用了。给ipad恢复系统恢复软件已经有不少经验了，这到难不到我，倒是苹果低版本的备份恢复到高版本时出错搞得郁闷。这几天也许日子比较好，想要的软件都降价了，全收了进来，我也成了正版使用者，越狱貌似和我关系不大了。

装ubuntu11.10，纠结了很就一直在犹豫要不要装，最后经过痛苦的挣扎还是装吧，这次不搞双系统了，直接干掉win7，电脑没光驱要准备的事情还挺多，全盘备份，制作U盘启动盘等等。经过一翻折腾，总算到能用的状态，不过还是百密一疏，刚改好的文章没备份下来，装系统的细节就重写篇文章再说。

折腾高斯，想做完实验之余，从理论上在验证下。以前看师兄师姐用过，现在自己有了那么点想法，发现用起来并没有那么简单，虽说选选参数什么都很简单，但是选得合理可信就难了。没学过量子化学和物质结构，很多参数都不知道什么意思。还好在黄师兄的指导下明白了点，现在黄师兄已经是黄所长了。

做实验，不用说了，更是个累人的差事，鬼知道可行不，还要带着啥都不懂的干活，也许我刚来的时候也是这样，够让带我的老师头疼的，经过5年的磨练，我也开始带人了。

老板想法多，我上次只是随便说了下，他以为我都做完了，这个从无到有要多难，要是我生下来就会不用学多好。本来让我搞硬件做实验，现在又要搞软件算法，我真的要被培养成德智体美劳全面发展的五好青年了。这样也好，忙就忙吧，充实下自己，打发无聊的时间，多学些东西。

PS：手机码字，做实验的时候想到点写点，连贯性比较差，正好可以打发时间。

回归Linux准备

Oct 16th, 2011 | Comments

回归Linux准备

 

纠结了很久，到底换不换系统，win7用着也还可以，但是总觉得不是很爽，反正老板给的台机可以用XP，自己电脑上总可以随便折腾，我就再试一把吧，这次来个狠的，把整个硬盘都干掉重新来，彻底抛弃windows。不过在安装前要做很多准备工作，本来以为可以用科大的pxe安装的，备份资料发现不是科大校园内网用不了，我勒个去，对于我这没光驱的本子可麻烦大了。不过以我装过N次系统的经验，两手准备吧，先把镜像写到U盘上安装，如果不行再搞个移动光驱来装，应该可以成功的。

 

在装系统的第一步就是备份，这次来得太狠要格全盘，还好我有块500G硬盘还有比较大的空间，就把D盘全copy过去了，吃饭回来发现根本没动，杀毒软件觉得有电脑上有几个算号软件有问题，问我是不是删除还是隔离，白白浪费我两个小时，我为了节省时间，点全部删除，这样还是花了我一个多小是备份70多G的东西。C盘还有一些要备份，估计今天晚上没时间弄了，什么邮件、QQ聊天记录、收藏夹等等。

 

制作U盘镜像，找了个比较简单的图文教程，用ultraISO的，因为这个比较熟悉也简单。还有一个用软件写镜像写入grub的，看起来麻烦，我也懒得看，就试试能不能成功。别人都是用desktop制作的，我想我是要安装系统就不搞桌面版，就直接搞alternate版本的制作，结果从U盘启动成功了，看着久违的安装界面还不能安装，真是心痛，明天把资料备份好就装。Ubuntu11.10我来了！

合肥半日游

Oct 16th, 2011 | Comments

昨近烦心的事情太多，早就想骑车出去逛逛，正好遇到院里停水又停电，逼得我们这些宅男没地方去，只好骑车出去游玩。本来的计划是到Tan老师家舒城那边转转，早上花三到四个小时到舒城，下午去万佛湖转转，第二天返回。可是他事情太多，而且车又出了点问题，临时改变了计划。

 

由于晚上玩得比较晚，所以早上起不来，Tan老师给我打了两个电话才出门。天气不错，出门穿了件google的T恤和一件抓绒衣，下午可把我热死了。从我住的地方到Tan老师的地方尾随一电动车跑了2/3的路程，无奈风太大又没早饭，在最后一个上坡的时候实在没力气眼睁睁的看着它跑了。刚到Tan老师楼下拿出手机，他又打电话来，真是巧了。等的时候拿出手机给爱车拍照，才发现把SD卡放电脑里了，相机里40M的内存也照不了几张。

我的折叠车

先上张路线图

第一件事就是去兔子那买装备，顺着长江路到家乐福取钱，再绕环城路再金寨路到金粮巷买了个头盔。本来想买完东西到阿来那修车，打电话给阿来说12点才回家。我们总要找件事做，我本来计划着顺徽州大道到头去巢湖边看看，可他觉得太远了，我们就到明珠广场那里看文博会，感觉骑了好远才到明珠广场，风实在太大，我这体重都快被吹倒了。我们到那里时候文博会开幕什么都结果了，也没什么好看的，顺便就在拍了张照片，很多人来看我们的车，挺不好意思就走了，因为没吃早饭，看到蒙古大串太想吃，10块钱三串要了9串，吃得又恶心又渴，花了8块钱买了可乐，Tan老师一口气就干完了，一看就是没什么骑车的经验。

上张自暴

看时间已经11点，虽然已经知道阿来的住处，可是前两次都是打去，现在只能摸过去。问Tan老师不知道怎么走，只好我带路了，早拐了一个路口，就多绕了几公里路，也还不错，到那时候阿来已经在家了。把车牙盘拆掉清洗一翻，才发现根本就没毛病，主要是车洗太干净没油了。我在旁边折腾了一下边上几辆豪车，后来说给他听，又诱惑了他一翻，听得他直流口水。

 

从阿来那出来12点多，晚上才来电，这样就回去太早了，所以我提议他去爬又蜀山，他是不愿意骑车上的，最后还是被我逼去。回去路上西风实在太大，根本就骑不动，也许中午没吃饭没力气了。到山脚补给水份，然后上山，和自己上次骑车上山已经有1年有余了，实在骑不动，就先在平台处歇会，骑了一半Tan老师累得上气不接下气，于是又休息，不知道磨了多久就到了山顶。不过下山可爽了，但是为了保证Tan老师的安全，我还是放慢了速度，下来才发现他刹车没有调好，真得吓多一跳，要是出事就麻烦了。

Tan老师带头盔后的自然风发型

下山后才3点多，回去又太早，就到网吧找同学，多年没去网吧的我们实在不太习惯那里恶心的环境，玩了一会就走了。实在没事做就和同学到Tan老师那约了同学打实况，同学风头正盛的时候被我打得找不着北，不过在加赛时间还是金球赢了我，搞得我挺不爽。晚饭Tan老师包了，我们又去北京宝顺吃火锅，这几天肚子不好，点了些清淡的，也没什么胃口，吃了点金针菇就不吃了。我们饭价超过200了，送了10瓶酒，一个一瓶喝了3瓶还有7瓶，我又骑着车要摇摇晃晃带着7瓶酒回家，总共凑齐10瓶酒。

 

又过了有意义的一天，经过大半天的骑行，发现小车比大车操控，轴得润度都差太多，唯一好处就是体积小，重量轻，很适合通勤使用，那大车就运动用吧。

PS：今天总里程加上上山的3公里共70公里左右。
